# Technological Stack
To carry out the complete development of the project, different tools will be used in each phase

## Frontend
In this phase we will use the following technologies:

- HTML5

- CSS

- Bootstrap

- UIKIT

- JAVASCRIPT

- Fetch

- Thymeleaf

## Backend
In this phase we will use the following technologies:

- Java 11

- Hibernate

- Spring Boot

- Spring Data

- Spring MVC y JPA

- Spring Security

- HQL

- CriteriaBuilder

## Database
To store user data we will use:

- MariaDB

## Management and documentation
For the management and documentation of the project we will work with:

- Git

- GitHub

- Docker

Finally, as a long-term goal, we would implement a new section for managing a business.
